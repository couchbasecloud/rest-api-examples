 #  listUsers
Returns , in a table format, a list of Couchbase Cloud Users

The table shows the   name, email address, and unique ID within Couchbase Cloud

Example
```
 Users
---------------------------------------------------------------------------
|     Name      |      Email       |                  ID                  |
---------------------------------------------------------------------------
| Fred Smith    | fsmith@acme.com  | 32cf63e0-b65f-46c0-b936-41542e5ac768 |
---------------------------------------------------------------------------



```
